# MyKeyboardTyping

自作キーボード向けタイピング練習ツール

🌐 **https://MiddleSouth.github.io/MyKeyboradTyping/**

## 概要

**あなたのキーボード配列そのままで、タイピング練習ができます。**

MyKeyboardTypingは、WebHID APIを使用してキーボードから直接キーマップを読み込み、画面に表示するタイピング練習ツールです。どんなにカスタマイズされたキーボードでも、実際の配列を見ながら効率的に練習できます。

### なぜ必要か

タイピング練習は、キーボードの画像で配置を確認しながら行うのが最も効率的です。しかし、既存のタイピング練習ツールは通常のキーボード配列を前提としており、自作キーボードユーザーには適していません。

**自作キーボード特有の課題**:

- カラムスタッガードや分割配列など、通常のキーボードとは配置が大きく異なる
- ユーザーごとにキーマップを大幅にカスタマイズしている
- 練習ツールに表示される標準配列が、実際のキーボードと一致しない

### 解決方法

このツールは、**あなたのキーボードから直接キーマップを読み込み**、画面に表示します。これにより、どんなにカスタマイズされたキーボードでも、実際の配列を見ながら最適なタイピング練習が可能になります。

現在はErgo68に対応しており、将来的には他の自作キーボードにも対応を拡大する予定です。

## 主な機能

### キーボード連携

- 自動キーマップ読み込み: WebHID APIで接続したキーボードのキーマップを自動取得
- 実配列表示: あなたのキーボードの配列がそのまま画面に表示される
- 複数レイヤー対応: レイヤー0〜3まで切り替えて表示可能
- リアルタイムハイライト: 押したキーがキーボード画像上で即座にハイライト

### 練習機能

- 日本語タイピング: ローマ字入力によるひらがな練習
  - 複数入力パターン対応（「し」→ si/shi など）
  - 拗音・促音対応（きゃ、っ など）
  - 動的パターン切り替え
- プログラミング練習: C#コードを使った記号・キーワード練習
- 統計表示: 正確率、正答数、誤答数をリアルタイム表示

## 対応環境

- ブラウザ: Chrome/Edge（WebHID API対応ブラウザ）
- OS: Windows （macOS, Linuxでも動くかもしれませんが、動作確認はしていません）
- キーボード: Ergo68
  - 将来的には全てのHID対応キーボードに対応する仕組みを構築したいです

## 使い方

### 1. キーボードの接続

1. ツールにアクセスすると、キーボード選択画面が表示されます
2. 「キーボードを選択」ボタンをクリック
3. ブラウザのデバイス選択ダイアログで、接続されているキーボードを選択
4. キーボードが認識されると、キーマップが自動的に読み込まれます

### 2. 練習素材の選択

キーボード選択後、以下の練習素材から選択できます

#### 日本語練習

- ひらがな基礎: あいうえお〜ぱぴぷぺぽ
- 拗音・促音: きゃきゅきょ、っ の練習
- 日常単語: よく使う日本語の単語20個

#### プログラミング練習（C#）

- 基本記号: `{}`, `[]`, `=>` など
- キーワード: class, public, async など
- コード例: 実践的なC#コードスニペット

### 3. タイピング練習

1. 練習素材を選択すると練習画面に遷移します
2. 画面上部にひらがな（日本語の場合）またはテキストが表示されます
3. 画面下部にローマ字入力が表示されるので、その通りに入力します
4. キーを押すと、キーボード画像上で該当キーがハイライトされます
5. 正しく入力すると緑色、誤入力すると赤色で表示されます
6. 1単語完了すると自動的に次の単語に進みます

### 4. 結果の確認

全ての単語を入力し終えると、完了画面が表示されます

- 正答数: 正しく入力できた文字数
- 誤答数: 誤って入力した回数
- 正確率: 正答数 ÷ (正答数 + 誤答数) × 100

完了画面では以下の操作が可能です

- BackSpaceキー: 同じ練習を最初からやり直す
- Enterキー: 次の練習素材を選ぶ画面に戻る

## 日本語入力のポイント

### 複数パターン対応

同じひらがなに対して複数の入力方法が使えます

| ひらがな | 入力方法 |
|---------|---------|
| し | si または shi |
| ち | ti または chi |
| つ | tu または tsu |
| ふ | hu または fu |

画面には最初のパターンが表示されますが、別のパターンで入力を始めると自動的に切り替わります。

### 促音（っ）の入力

- 基本: 次の文字の子音を重ねる
  - 「きって」→ `kitte`
  - 「がっこう」→ `gakkou`

- 単独の場合: 直接入力
  - 「っ」→ `ltu` または `xtu`
